# Legal Documents

This directory stores example source documents used for ingestion and retrieval experiments.

Current example:

- `digital-services-act-en.pdf`

Source:

- EUR-Lex official publication for Regulation (EU) 2022/2065
- CELEX: `32022R2065`
- English PDF URL:
  `https://eur-lex.europa.eu/legal-content/EN/TXT/PDF/?uri=CELEX:32022R2065`

Why this document:

- official legal source
- long enough for meaningful chunking and retrieval
- structured with recitals, articles, definitions, obligations, and enforcement sections
